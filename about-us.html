The Art of Web Design: My Journey Through ITSE 1301 - 22700 Web Design Tool

In the ITSE 1301 - 22700 Web Design Tools class, I gained valuable skills and knowledge that have helped me become proficient in web development. Through this class, I learned how to use FileZilla to upload and manage files, organize files and directories in a logical and efficient manner, and create HTML forms and tables to structure and present data. In addition, I became skilled in using CSS to style and format web pages, including understanding the box model and floating elements to create appealing layouts. I also learned how to use the w3.css framework to further enhance my designs, and how to integrate icons to add visual interest. To ensure the quality and functionality of my websites, I learned how to use a template to streamline my design process and make my websites more consistent. Furthermore, I became proficient in using the validator service to check for HTML and CSS bugs in my code, ensuring that my websites are optimized for speed and reliability. Overall, the ITSE 1301 - 22700 Web Design Tools class has equipped me with valuable skills that I will continue to use and build upon in my future web development projects. These skills are not only useful for personal projects, but also for pursuing a job in software development. By demonstrating my competence in web development, I can showcase my ability to create user-friendly and attractive websites that meet the needs and expectations of clients and users. Moreover, by learning how to use various tools and frameworks, I can adapt to different technologies and platforms that are relevant in the software industry. Therefore, the ITSE 1301 - 22700 Web Design Tools class has prepared me for a successful career in software development.
